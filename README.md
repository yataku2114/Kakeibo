# しんぷる家計簿

## サイト概要
誰でも簡単に使える家計簿です。

### サイトテーマ
月々のお金を見える化

### テーマを選んだ理由
お金を見える化することで、お金を管理できるようになる。
そうすることで、将来に向けたお金の考えもできる。

### ターゲットユーザ
お金を管理したい人

### 主な利用シーン
入出金があった時にサイトに入力するだけ

## 機能一覧
・ユーザー登録、ログイン機能（devise）
・収入、固定、変動費科目登録機能
・各項目の金額登録機能
・各月の収支差表示機能
・要望機能

## 開発環境
- OS：Linux(CentOS)
- 言語：HTML,CSS,JavaScript,Ruby,SQL
- フレームワーク：Ruby on Rails
- JSライブラリ：jQuery
- IDE：Cloud9
